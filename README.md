# CsvFebrabanReader
Leitor de Faturas Csv Febraban

## Installation

    $ bundle

Or install it yourself as:

    $ gem install csv_febraban_reader

## Usage

    $ rake
    $ CsvFebrabanReader.detalhes('/path/to/file')
    $ CsvFebrabanReader.detalhesByPhone('/path/to/file', "xx-xxxx-xxxx')
